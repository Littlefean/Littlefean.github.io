C语言实现超级大数的加法、乘法、阶乘

说明

超级大数的范围在 0~1e9223372036854775807，也就是最大是 99999........99999（中间9223372036854775807个9）

实现了：

  • 大数 + 大数 => 大数
  • 大数 x 个位数 => 大数
  • 大数 x 大数 => 大数
  • int阶乘 =>大数

所有参与运算的大数都要大于0，还没有做小于0的处理。

主要实现原理和算法思路

用数组表示大数，每一位表示一个数字。

加法就是将两个大数按位累加，处理进位就用一个额外的变量保存。

乘法 a x b 就是将b的每一位数字和a相乘（每一位都处理和判断进位问题），对应还要将它扩大10的n次方倍，再将所有的相乘的结果相加构成一个新的大数。就是结果。（和小学手算高位数乘法，列竖式，原理一样）

阶乘：不停的调用大数乘法。

缺点

用数组实现的大数，但是数组里面的每一位都用的一个int来表示一个个位数字，空间浪费严重。

时间复杂度可能依然待优化，计算6000的阶乘需要6秒左右。

使用了数组动态扩容的方法来存储大数，没有考虑到链表这种节点式存储结构

还没有实现负数，减法操作，因此只能表示正数

潜在的bug和漏洞

将字符串转成大数的过程中没有处理非数字字符，如果将含有非数字字符的字符串转化成大数，会出现错误。

大数销毁方法的好像还不干净彻底，经过测试发现循环打印1~1000的阶乘，内存占用会持续增长，累计占用达到了8G左右。
